# 🇮🇱 סורק מניות ת"א — שיטת מרפי
## TASE Israel Stock Scanner (Murphy Method)

סורק מניות לבורסת תל אביב המבוסס על שיטת הניתוח הטכני של ג'ון מרפי.  
מחפש **מניות oversold בתוך מגמת עלייה** בשוק הישראלי.

---

## 🧠 לוגיקת הסריקה (מותאמת לשוק הישראלי)

| קריטריון | כלל |
|----------|-----|
| **מחיר** | > ₪10, מעל MA200 (או MA120 אם אין מספיק היסטוריה) |
| **נפח** | ממוצע יומי > 100,000 מניות |
| **בטא** | ≥ 0.5 (מחושב מול ת"א 125) |
| **RSI(14)** | < 55 — מחפשים oversold |
| **BB%B** | < 0.40 — מחיר ליד רצועה תחתונה |
| **מגמה 4 שבועות** | חיובית |
| **מגמת 52 שבועות** | מחיר > שנה קודמת + MA50 > MA200 |

## 🏅 ניקוד (0–10)

- עומק RSI (oversold)
- מיקום BB (ליד רצועה תחתונה)
- חוזק מגמה 4 שבועות
- יישור ממוצעים נעים
- ספייק נפח
- חוזק יחסי מול ת"א 125
- קרבה לרמת תמיכה
- תבניות גרף (תחתית כפולה, משולש עולה)
- יחס R/R

## 📊 בק-טסט

- מוריד 12 חודשים של נתונים יומיים
- **קנייה:** RSI < רף + BB%B < 0.40 + מחיר > MA ארוך
- **מכירה:** RSI > 65 OR BB%B > 0.80 OR מחיר < MA50 × 0.95
- מחשב: % הצלחה, תשואה ממוצעת, ימי החזקה
- מריץ רק על מניות שעברו את שלב 1
